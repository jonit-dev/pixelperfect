You've sent an image to the printer only to receive a blurry, pixelated mess. Or worse—the print shop rejected your file entirely. Understanding image resolution for printing isn't intuitive, but it's essential for anyone who needs physical prints.

This guide explains DPI, resolution requirements, and how to ensure your images print beautifully at any size.

## Understanding DPI vs PPI

First, let's clarify the terminology that confuses everyone:

**PPI (Pixels Per Inch)** - Describes digital images. How many pixels fit in one inch on screen.

**DPI (Dots Per Inch)** - Describes printed output. How many ink dots the printer places per inch.

In practice, most people use "DPI" for both. When preparing files for print, you're really setting the PPI, but print shops call it DPI. We'll use DPI throughout this guide since that's what you'll encounter in the real world.

## The Golden Rule: 300 DPI

For most printing purposes, **300 DPI** is the standard:

| Print Type | Recommended DPI | Minimum DPI |
|------------|-----------------|-------------|
| Photo prints | 300 | 200 |
| Magazines | 300 | 300 |
| Brochures | 300 | 250 |
| Business cards | 300 | 300 |
| Posters (close viewing) | 300 | 200 |
| Billboards | 30-100 | 20 |
| Banners (far viewing) | 100-150 | 75 |

<Callout type="info">
The further away viewers will be, the lower DPI you can use. A billboard viewed from 50 feet away looks fine at 30 DPI. A photo held in your hands needs 300 DPI.
</Callout>

## Calculating Required Pixels

Here's the formula you need:

```
Required Pixels = Print Size (inches) × DPI

Example: 8×10 inch print at 300 DPI
Width: 8 × 300 = 2400 pixels
Height: 10 × 300 = 3000 pixels
```

### Common Print Size Requirements

| Print Size | At 300 DPI | At 200 DPI | At 150 DPI |
|------------|------------|------------|------------|
| 4×6" | 1200×1800 | 800×1200 | 600×900 |
| 5×7" | 1500×2100 | 1000×1400 | 750×1050 |
| 8×10" | 2400×3000 | 1600×2000 | 1200×1500 |
| 11×14" | 3300×4200 | 2200×2800 | 1650×2100 |
| 16×20" | 4800×6000 | 3200×4000 | 2400×3000 |
| 24×36" | 7200×10800 | 4800×7200 | 3600×5400 |

### Quick Calculation Tool

To find your maximum print size from an existing image:

```
Max Print Size = Pixels ÷ DPI

Example: 4000×3000 pixel image
At 300 DPI: 4000÷300 = 13.3" × 3000÷300 = 10"
Maximum quality print: 13×10 inches
```

## When Your Image Is Too Small

This is where most people get stuck. You have a 1200×800 pixel image and need a 16×20 inch print.

**The math:**
- Required: 4800×6000 pixels (at 300 DPI)
- Available: 1200×800 pixels
- Shortfall: 4x upscale needed

### Option 1: Accept Lower Quality

Print at lower DPI:
- 1200÷16 = 75 DPI

At 75 DPI, prints will be noticeably soft. Only acceptable for:
- Large posters viewed from distance
- Draft/proof prints
- Non-critical applications

### Option 2: Print Smaller

Calculate your true maximum size:
- 1200÷300 = 4 inches (width)
- 800÷300 = 2.7 inches (height)

Your image can only print at 4×2.7 inches at full quality.

## Preparing Files for Print

### Step 1: Check Your Current Resolution

In most image editors:
- **Photoshop**: Image → Image Size
- **GIMP**: Image → Scale Image
- **Preview (Mac)**: Tools → Adjust Size

Note both pixel dimensions and current DPI setting.

### Step 2: Calculate Requirements

Use the formulas above to determine if your image is sufficient.

### Step 3: Upscale If Needed

If pixels are insufficient:

1. **Upscale first** using AI enhancement
2. **Then set DPI** in your image editor
3. **Don't resize and set DPI simultaneously**—this causes quality loss

### Step 4: Set Correct DPI

In Photoshop:
1. Image → Image Size
2. Uncheck "Resample"
3. Set Resolution to 300 pixels/inch
4. Dimensions will update to show print size

<Callout type="warning">
If "Resample" is checked, Photoshop will add/remove pixels when you change DPI. Keep it unchecked to just change the metadata without affecting pixel count.
</Callout>

### Step 5: Convert Color Mode

Print uses CMYK color, not RGB:
- **RGB**: Screens (red, green, blue light)
- **CMYK**: Print (cyan, magenta, yellow, black ink)

For professional printing:
1. Convert to CMYK before sending
2. Proof colors—some RGB colors can't be reproduced in print
3. Ask your printer for their color profile

For consumer photo printing (Walmart, CVS, online services):
- RGB is usually fine—they handle conversion
- sRGB color space is safest

### Step 6: Save in Correct Format

| Format | Use For | Notes |
|--------|---------|-------|
| TIFF | Professional print | Lossless, large files |
| PDF | Documents, posters | Preserves vectors and fonts |
| JPEG | Photos | Quality 90+ for print |
| PNG | Graphics with transparency | Not ideal for photos |

## Special Printing Scenarios

### Canvas Prints

Canvas has visible texture that hides some detail:
- 150-200 DPI is often sufficient
- Smaller sizes still benefit from 300 DPI
- Gallery wraps need extra margin for edges

### Metal/Acrylic Prints

These show extreme detail:
- 300 DPI minimum recommended
- Higher is better—surface is very smooth
- Imperfections are magnified

### Large Format (Posters/Banners)

Viewing distance matters more than DPI:

| Viewing Distance | Minimum DPI |
|-----------------|-------------|
| 1-2 feet | 200-300 |
| 3-5 feet | 150-200 |
| 6-10 feet | 100-150 |
| 10+ feet | 50-100 |

### Fine Art Prints

Galleries and exhibitions demand highest quality:
- 300 DPI minimum, 360 preferred
- Giclée printers can use 360+ DPI
- Paper quality matters as much as resolution

## Troubleshooting Print Issues

### "Image Resolution Too Low" Error

The print service is rejecting your file because pixels are insufficient.

**Solution**: Upscale your image before uploading.

### Colors Look Different in Print

RGB to CMYK conversion changed colors.

**Solution**:
- Request a proof print first
- Use printer's color profile
- Avoid extremely saturated colors

### Print Looks Blurry

Image was stretched beyond its resolution.

**Solution**:
- Calculate true maximum print size
- Upscale with AI before printing
- Choose smaller print size

### Banding in Gradients

Not enough color information for smooth transitions.

**Solution**:
- Use 16-bit files if possible
- Add slight noise to break up banding
- Request higher quality printing
